# UI设计规范

## 核心原则
采用极简专业的技术博客设计理念，100%以内容为核心，完全弱化装饰性元素，极致优化长文阅读体验。

## 配色系统
- 主背景：纯白色（#FFFFFF）/ 极深灰（#121212，暗色模式）
- 文字：主文本使用深灰（#1a1a1a）/ 浅灰（#e0e0e0，暗色模式）
- 强调色：仅使用1种低饱和度主色（深蓝/深绿系）用于链接、高亮和交互元素
- 分割线/边框：使用极浅灰（#f0f0f0）/ 深灰（#2a2a2a，暗色模式），线条粗细不超过1px

## 排版系统
- 字体层级：标题使用优雅衬线字体，正文使用高可读性无衬线字体
- 行高：正文行高1.6-1.75，标题行高1.2-1.3
- 字号：正文字号16-18px（桌面端），15-16px（移动端）
- 内容区域最大宽度限制在700-800px

## 交互与体验
- 悬停效果：极简，仅文字颜色轻微变化或下划线
- 动画：所有交互动画时长不超过200ms
- 支持暗色/亮色模式自动切换
