# Next.js 16 入门指南

Next.js 16 带来了许多令人兴奋的新特性，让我们一起探索！

## App Router

App Router 是 Next.js 16 的核心特性之一，提供了更直观的文件系统路由：

- 使用 `app/` 目录替代 `pages/`
- 支持嵌套布局和路由组
- 内置加载状态和错误处理

## React 19 集成

Next.js 16 与 React 19 深度集成，带来性能提升：

- 更快的服务器组件
- 优化的客户端导航
- 改进的开发体验

## 性能优化

- Turbopack 作为默认构建工具
- 自动图片优化
- 静态页面预渲染
