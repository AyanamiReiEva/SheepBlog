# Git提交规范

## 提交信息格式

```
<type>(<scope>): <subject>

<body>

<footer>
```

## Type类型

- feat: 新功能
- fix: 修复bug
- docs: 文档更新
- style: 代码格式调整
- refactor: 重构
- test: 测试相关
- chore: 构建/工具相关

## 示例

```
feat(blog): add dark mode toggle

- Add theme provider component
- Add dark/light mode switch in header
- Persist theme preference in localStorage

Closes #123
```
