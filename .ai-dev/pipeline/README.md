# AI开发流水线配置

## 流水线阶段

### 1. 需求分析阶段
- 接收自然语言需求
- 调用需求分析师智能体
- 生成结构化需求文档
- 需求合法性校验

### 2. 代码开发阶段
- 任务拆解与分配
- 前端/后端智能体开发
- 代码自动写入
- 架构师代码评审

### 3. 测试阶段
- 单元测试自动生成
- 测试运行与覆盖率报告
- E2E测试生成与执行
- 失败反馈与修复

### 4. 合规校验阶段
- ESLint/Prettier自动检查
- 代码质量扫描
- 安全漏洞扫描
- 合规报告生成

### 5. 部署阶段
- 自动创建PR
- 自动合并代码
- Vercel自动部署
- 健康检查与通知

## 使用说明

1. 配置Harness CI/CD环境
2. 导入base.yaml流水线模板
3. 配置GitHub Webhook触发器
4. 配置环境变量（API密钥等）
5. 提交需求触发流水线
